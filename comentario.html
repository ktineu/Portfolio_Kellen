Este site foi desenvolvido por Kellen Tineu Camargo e tem o objetivo de apresentar o que foi aprendido na formação front-end dos cursos online de tecnologia.

Foram abordados conceitos em HTML - linguagem de marcação, usado para estruturar páginas web; CSS - linguagem de estilo, que foca no design e aparência do site e o JavaScript - linguagem de programação, que permite a criação de conteúdos dinâmicos, interativos e atraentes.

O Portfólio encontra-se disponível no repositório do Github em http://github.com/ktineu/Portfolio_Kellen

Portanto, a empresa "Kellen T - Salão de Beleza", é fictícia , sendo uma inspiração somente para a criação deste Portfólio com a finalidade de integrar teoria e prática no desenvolvimento de websites utilizando as principais tecnologias web.

Agradeço a todos os envolvidos no processo de ensino tech.
